# 宝塔面板部署指南

## 服务器信息
- **IP**: 81.70.191.44
- **端口**: 3000

---

## 一、准备工作

### 1. 上传代码
将项目文件上传到服务器：
```bash
/www/wwwroot/memory-ai-companion/
```

### 2. 安装 Node.js
1. 登录宝塔面板
2. 进入 **软件商店**
3. 搜索 **Node.js**
4. 安装 **Node.js 版本管理器**
5. 安装 **Node.js v18** 或 **v20**

---

## 二、部署后端服务

### 1. 进入项目目录
```bash
cd /www/wwwroot/memory-ai-companion/server
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置环境变量
创建 `.env` 文件：
```bash
cd /www/wwwroot/memory-ai-companion/server
nano .env
```

添加以下内容：
```env
PORT=3000
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
NODE_ENV=production
```

**注意**：将 `JWT_SECRET` 替换为一个复杂的随机字符串！

### 4. 初始化数据库
```bash
node init-db.js
```

---

## 三、启动服务（推荐 PM2）

### 1. 安装 PM2
```bash
npm install -g pm2
```

### 2. 启动服务
```bash
cd /www/wwwroot/memory-ai-companion
pm2 start ecosystem.config.js
```

### 3. 保存配置
```bash
pm2 save
pm2 startup
```

### 4. 常用命令
```bash
# 查看状态
pm2 status

# 查看日志
pm2 logs memory-ai-companion

# 重启服务
pm2 restart memory-ai-companion

# 停止服务
pm2 stop memory-ai-companion

# 删除服务
pm2 delete memory-ai-companion
```

## 四、配置 Nginx 反向代理（可选）

### 1. 添加网站
1. 宝塔面板 → **网站**
2. 点击 **添加站点**
3. 填写域名（如果没有域名，填写服务器 IP）
4. 根目录选择 `/www/wwwroot/memory-ai-companion`

### 2. 配置反向代理
1. 点击网站 → **设置**
2. 选择 **反向代理**
3. 添加反向代理：
   - 代理名称：`api`
   - 目标 URL：`http://127.0.0.1:3000`
   - 发送域名：`$host`

### 3. 或者手动配置 Nginx
在宝塔面板的网站配置中添加：

```nginx
location /api/ {
    proxy_pass http://127.0.0.1:3000/api/;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_cache_bypass $http_upgrade;
}
```

## 八、常见问题

### 1. 端口被占用
```bash
# 查看端口占用
netstat -tulpn | grep 3000

# 杀掉占用进程
kill -9 <PID>
```

### 2. 权限问题
```bash
# 修改目录权限
chmod -R 755 /www/wwwroot/memory-ai-companion
chown -R www:www /www/wwwroot/memory-ai-companion
```

### 3. 数据库权限
```bash
# 确保数据库文件可写
chmod 666 /www/wwwroot/memory-ai-companion/server/database.sqlite
```

### 4. 查看错误日志
```bash
# PM2 日志
pm2 logs memory-ai-companion

# 或者查看日志文件
tail -f /www/wwwroot/memory-ai-companion/logs/err.log
tail -f /www/wwwroot/memory-ai-companion/logs/out.log
```

## 九、SSL 证书配置（HTTPS）

### 1. 申请证书
1. 宝塔面板 → **网站**
2. 点击网站 → **SSL**
3. 选择 **Let's Encrypt** 或上传自己的证书

### 2. 强制 HTTPS
在 Nginx 配置中添加：
```nginx
server {
    listen 80;
    server_name 81.70.191.44;
    return 301 https://$server_name$request_uri;
}
```
