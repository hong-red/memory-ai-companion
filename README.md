# 小世界·记忆 - AI 伴侣

一个温暖、个性化的 AI 情感陪伴应用，支持智能对话、语音交互、日记记录等功能。

## ✨ 主要功能

### 💬 智能对话
- 基于 Moonshot AI API 的自然对话
- 支持自定义 AI 角色性格
- 长期记忆，记住你们的聊天内容

### 🎤 语音交互
- 语音输入，解放双手
- 可选语音回复

### 📔 智能日记
- 手动记录日记，选择心情
- AI 自动总结聊天记录生成日记
- 连续记录天数统计

### 🎭 个性角色
- 内置多个预设角色（叶修、助手等）
- 支持创建自定义 AI 角色
- 可上传 AI 专属头像

### 🎨 界面个性化
- 自定义聊天背景图片
- 用户和 AI 头像设置
- 精美的聊天气泡动效

## 🚀 快速开始

1. 克隆项目
```bash
git clone https://github.com/hong-red/memory-ai-companion.git
```

2. 打开 `index.html` 文件即可使用

3. 在设置页面配置 Moonshot API Key

## ⚙️ 配置说明

需要申请 [Moonshot AI](https://platform.moonshot.cn/) 的 API Key 才能使用聊天功能。

## 🛠 技术栈

- 纯前端 HTML/CSS/JavaScript
- LocalStorage 本地数据存储
- Web Speech API 语音识别
- Moonshot AI API

## 📱 特性

- 💯 纯前端，无需后端服务器
- 📱 响应式设计，支持移动端
- 🌐 支持 PWA 安装
- 🔒 数据本地存储，保护隐私
